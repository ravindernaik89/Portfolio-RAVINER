## File Descriptions

### Root Files
- **index.html**: The main entry point of the website containing all sections

### CSS Folder
- **styles.css**: Contains all the styling for the portfolio
- **responsive.css**: Media queries for mobile/tablet responsiveness
- **animations.css**: Separate file for animation keyframes and effects

### JS Folder
- **main.js**: Core JavaScript functionality (scroll effects, navigation, etc.)
- **animations.js**: Handles scroll-triggered animations
- **form-validation.js**: Contact form validation logic (if you add a form)

### Images Folder
- **profile.jpg**: Your professional headshot
- **hero-bg.jpg**: Optional background image for hero section
- **projects/**: Screenshots and demos of your projects
- **icons/**: Social media and UI icons

### Assets Folder
- **fonts/**: Custom web fonts
- **documents/**: Downloadable files like your resume

### Docs Folder
- **DEPLOYMENT.md**: Instructions for deploying to GitHub Pages, Netlify, etc.

## How to Use

1. **Add your content**: 
   - Place your profile photo in `/images/`
   - Add project screenshots to `/images/projects/`
   - Put your resume PDF in `/assets/documents/`

2. **Customize**: 
   - Edit `index.html` with your information
   - Modify `css/styles.css` for design changes
   - Update colors, fonts, and layout as needed

3. **Deploy**:
   - Upload to GitHub Pages
   - Deploy to Netlify or Vercel
   - Host on any web server
